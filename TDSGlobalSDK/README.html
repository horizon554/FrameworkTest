TDS-iOS

1.接入前准备

1.1 TDSSDK说明

TDSSDK支持登录和支付功能，最低iOS版本为iOS10.以下framework为静态库。

TDSSDKCoreKit.framework     TDSSDK核心库
TDSSDKLoginKit.framework    TDSSDK登录组件
TDSSDKIAPKit.framework      TDSSDK支付组件

1.2 添加TDSSDK

1.2.1 添加TDSSDK文件

  • 将TDSSDKCoreKit.framework和TDSSDKLoginKit.framework添加到项目，确保Link Binary With Libraries中已添加。

  • 将TDSDKResources.bundle添加到项目，确保Copy Bundler Resources中已添加

  • 支付组件TDSSDKIAPKit.framework视游戏需要添加

  • 由于第三方组件中有使用Swift开发部分，纯Objective-C项目需要打开Swift编译器，可在项目中添加Swift文件。将Blank.swift添加到项目

1.2.2 添加libs文件

  • 将libs中所有framework文件添加到项目，确保Link Binary With Libraries中已添加。

  • 将libs中所有bundle文件添加到项目，确保Copy Bundler Resources中已添加

1.3 添加系统依赖库

请检查项目中是否已自动添加以下依赖项：

AdSupport.framework
LocalAuthentication.framework
AuthenticationServices.framework
SystemConfiguration.framework
Accelerate.framework
SafariServices.framework
Webkit.framework
libc++.tdb

1.4 配置编译选项

在Build Setting中的Other Link Flag中添加-ObjC

在Build Setting中的Always Embed Swift Standard Libraried设置为YES

1.5 配置URL Types

选择应用TARGETS，点击Info标签，展开URL Types。添加以下几项 URL Scheme.大括号内为游戏App对应参数，在开发者后台获取。

TapTap  :   tt{taptap-client-id}
Facebook:   fb{facebook-app-id}
Google  :   {google-client-id}

2.TDSSDK使用说明

2.1 TDSSDK配置

在AppDelegate.m中,引入头文件

#import <TDSSDKCoreKit/TDSSDK.h>

在以下接口中添加调用

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
    [TDSSDK application:application didFinishLaunchingWithOptions:launchOptions];
    ...
}

- (BOOL)application:(UIApplication *)application
        openURL:(NSURL *)url sourceApplication:(NSString *)sourceApplication
     annotation:(id)annotation {
    [TDSSDK application:application openURL:url sourceApplication:sourceApplication annotation:annotation];
    ...
}

- (BOOL)application:(UIApplication *)app openURL:(NSURL *)url options:(NSDictionary<UIApplicationOpenURLOptionsKey, id> *)options{
    [TDSSDK application:app openURL:url options:options];
    ...
}

- (BOOL)application:(UIApplication *)application continueUserActivity:(NSUserActivity *)userActivity restorationHandler:(void (^)(NSArray<id<UIUserActivityRestoring>> * _Nullable))restorationHandler {
    [TDSSDK application:application continueUserActivity:userActivity restorationHandler:restorationHandler];
    ...
}

- (void)application:(UIApplication *)application didReceiveRemoteNotification:(NSDictionary *)userInfo fetchCompletionHandler:(void (^)(UIBackgroundFetchResult))completionHandler {
    ...
    [TDSSDK application:application didReceiveRemoteNotification:userInfo fetchCompletionHandler:completionHandler];
}

如果您的应用使用Scene管理App生命周期，在以下接口中添加调用

- (void)scene:(UIScene *)scene openURLContexts:(NSSet<UIOpenURLContext *> *)URLContexts API_AVAILABLE(ios(13.0)){
    [TDSSDK scene:scene openURLContexts:URLContexts];
}

- (void)scene:(UIScene *)scene continueUserActivity:(NSUserActivity *)userActivity  API_AVAILABLE(ios(13.0)){
    [TDSSDK scene:scene continueUserActivity:userActivity];
}

然后在合适的地方调用TDSSDK初始化

[TDSSDK initTDSSDK:^(BOOL success) {
    if (success) {
        NSLog(@"Init success");
    }else {
        NSLog(@"Init fail");
    }
}];

2.1 登录

引入头文件

#import <TDSSDKLoginKit/TDSLoginManager.h>
#import <TDSSDKCoreKit/TDSUserData.h>

调用以下接口

// 调用登录
[TDSLoginManager login:^(TDSUserData * _Nonnull result, NSError * _Nonnull error) {
    NSLog(@"result:%@,error:%@",result,error);
    if (error) {
        // login fail;
    }else {
        // login success;
    }
}];

// 监听用户状态改变
[TDSLoginManager addUserStatusChangeCallback:^(TDSUserStateChangeCode userStateChangeCode) {
    if (userStateChangeCode == TDSUserStateChangeCodeBindStateChange) {
        // user bind state change
    }else if (userStateChangeCode == TDSUserStateChangeCodeLogout) {
        // user logged out
    }
}];

// 打开用户中心
[TDSLoginManager userCenter];

// 登出当前用户
[TDSLoginManager logout];

2.2 支付

引入头文件

#import <TDSSDKIAPKit/TDSInAppPurchaseManager.h>

调用以下接口

[TDSInAppPurchaseManager payWithProduct:@"orderId" productId:@"productId" roleId:@"roleId" serverId:@"serverId" ext:@"ext" completionHandler:^(NSDictionary * _Nonnull orderInfo, NSError * _Nonnull error) {
    if (error) {
        // pay fail, handler error
    }else {
        // pay success
    }
}];

2.3 客服反馈

引入头文件

#import <TDSSDKCoreKit/TDSSDK.h>

用户登录成功以后，可以使用客服反馈功能.

[TDSSDK report:@"serverId" roleId:@"roleId" roleName:@"roleName"];

3.可选功能

以下功能游戏根据需要进行调用

3.1 分享

#import <TDSSDKCoreKit/TDSShareManager.h>

支持分享图片与URL，调用接口：

[TDSShareManager share:TDSShareTypeFacebook url:@"https://www.taptap.com/" message:@"taptap" completeHandler:^(NSError * _Nullable error, BOOL cancel) {
    if (error) {
        // fail
    }else {
        // success
    }
}];

3.2 数据跟踪

#import <TDSSDKCoreKit/TDSTrackManager.h>

支持简单数据追踪，调用接口：

[TDSTrackManager trackUser:@"roleId" roleName:@"roleName" serverId:@"serverId" level:0];
[TDSTrackManager trackEvent:@"eventName"];
[TDSTrackManager trackAchievement];
[TDSTrackManager eventCreateRole];
[TDSTrackManager eventCompletedTutorial];
